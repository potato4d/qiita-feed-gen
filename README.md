# Qiita Feed Generator

https://qiita-feed-gen.potato4d.me/

> Private feed generator for Qiita.

## Stacks

- Next.js
- Express
- Qiita API

## How to launch development server

```bash
$ yarn
$ yarn dev
```

## How to launch production server

```bash
$ yarn
$ yarn build
$ yarn start
```
